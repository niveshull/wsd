Posli, ki so sklenjeni v nasprotju s prepovedmi iz 2., 3. in 4. člena tega zakona, so nični.

Nični so tudi vsi posli, s katerimi v pravnih osebah, ki prirejajo posebne igre na srečo, nastaja delež zasebnega kapitala ali se zmanjšuje delež družbenega kapitala.

Št. 300-01/94-18/1 Ljubljana, dne 3. junija 1994
