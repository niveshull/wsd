Brigade so torej predstavljale »Rdečo tradicijo«. 
Četudi so se rodile iz mlade revolucionarne inteligence, so njeni člani prihajali iz vseh razredov in številni so bili navadni delavci. 
Neposredno po ugrabitvi nekdanjega premiera Alda Mora, leta 1978, je New York Times poročal: »Jedro Rdečih brigad sestavlja 400 do 500 »stalno zaposlenih« članov, ki so na plačilni listi organizacije. 
Njihova plača je 250.000 lir oziroma okoli 400 ameriških dolarjev na mesec, kar je minimalna plača italijanskega delavca …«
