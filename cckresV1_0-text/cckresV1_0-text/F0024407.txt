Nekoč v davnih časih so po vaseh in gozdovih strašile čarovnice. 
Imele so nadnaravne moči in so povzročale vse vrste neprijetnosti. 
Tako so npr. uročile žito, da ni raslo, živino, da je poginila, pa tudi nekatere moške so uročile tako, da niso več znali biti moški. 
Čarovnice so bile med seboj zelo različne. 
Nekatere so poznale skrivnosti zdravilnih rastlin in so s svojo modrostjo močno prekašale druge vaščane, še posebno župnike. 
Druge so bile premalo ali preveč lepe. 
Slednje so bile v zelo težkem položaju. 
Zaradi velikega povpraševanja so si nekatere privoščile bolj sproščeno spolno življenje, kot je bilo predpisano. 
Druge spet so se branile nezaželenih vsiljivcev, med katerimi se je našel tudi kakšen vaški ali cerkveni mogotec. 
Ti so že takrat težko prenesli, da jih je ženska zavrnila. 
Nekatere čarovnice so bile zgarane matere, ki svojim možem niso več zbujale poželenja ali pa so bile kratko malo preveč klepetave. 
Skratka, bile so odveč. 
Znano je tudi, da so nekateri čarovniki govorili stvari, ki jih nihče ni razumel. 
Eden od njih je celo trdil, da je zemlja okrogla, čeprav lahko vsak vidi, da je ploščata. 
Bili so tudi čarovniki, ki so podvomili v svetost svetih knjig. 
Bili so zelo odveč. 
Vladarji in cerkveni dostojanstveniki so enoglasno sklenili, da je treba ta zarod izkoreniniti. 
Izumili so inkvizicijo. 
Z vsemi silami so mučili čarovnice in čarovnike, dokler niso priznali greha, ki jim je podelil nadnaravne moči; spečali so se bili s hudičem in mu prodali dušo. 
Cerkveni dostojanstveniki so se čutili poklicani, da te nesrečne duše rešijo. 
To so napravili tako, da so na grmadi sežgali telesa, v katerih so prebivale. 
Hudič seveda spreminja podobo, kakor želi. 
Večinoma so se pojavljali moški hudiči, ki so dobili ime INCUBUS. 
Ženski hudiči so dobili ime SUCUBUS. 
Žrtve enih in drugih pa so dobile splošno »diagnozo«: ČAROVNIK ali ČAROVNICA. 
Seveda je bilo treba najti merila, po katerih bi ugotavljali čarovništvo. 
Tedaj še ni bilo laboratorijskih izvidov o navzočnosti hudičeve sperme, sploh pa je bilo vprašljivo, ali hudič potrebuje spermo. 
Zato so začeli iskati drugje. 
Hudič naj bi s posebnimi znamenji označil oziroma stigmatiziral telesa svojih žrtev. 
Sestavili so celo znanstveni priročnik z naslovom MALLEUS MALEFICARUM, ki je podrobno opisoval različne simptome in sindrome čarovništva. 
Potem se je svet nenadoma začel spreminjati. 
Nastale so tovarne, ki so potrebovale delovno silo. 
Ta je prihajala iz obubožanih kmetij.

Ko se spomnim nanj, se najprej spomnim na njegovo mačjo ljubezen. 
Naj vam opišem zgodbo. 
Johnny je imel poleg vseh lastnosti še eno veliko lastnost, namreč – požrešnost. 
Hrana ga je vedno zelo razveselila in prav nikoli se ni zgodilo, da bi jo v njegovi posodi po obroku še kaj ostalo. 
In kolikor se spominjam, nikoli ni prenašal – kakor se za psa tudi spodobi – mačk. 
Veselo je lajal nanje in jih preganjal naokoli, tako da sem se večkrat bala, da bo v spopadu s kakšno mačjo predstavnico ostal brez očesa. 
Nekoč pa je k nam na naš vrt zatavala potepuška breja muca. 
Kako sta se spoprijateljila, še danes ne vem.

Tudi skotila je in veselje je bilo nepopisno. 
Žal je čez nekaj časa poginila in Johnny jo je močno pogrešal. 
Še vedno je ob vsakem obroku klical muco k jedi, a ni več mogla priti.
