Kaj proučuje zgodovina

Spoznali smo že, da zgodovino ustvarjajo ljudje. 
In ljudi je že od nekdaj zanimala lastna preteklost. 
Pripovedovali in opisovali so dogodke, ki so jih doživeli ali so se zgodili njihovim prednikom. 
Tako se je razvila zgodovina – veda o preteklosti. 
Toda zgodovina dogodkov iz preteklosti ne le opisuje, ampak jih med seboj povezuje, ugotavlja njihove vzroke in posledice. 
Zgodovina proučuje razvoj človeštva od njegovega nastanka do danes. 
Kako je nastal človek? 
Kakšen je bil nekoč njegov odnos do okolja? 
Kako so si ljudje razlagali pojave v naravi? 
Kako jim je uspelo narediti velikanske piramide in templje? 
Ali so otroci tudi nekoč hodili v šolo? 
Kaj so otroci počeli, ko še ni bilo televizije? 
Kakšne so bile njihove igrače? 
Kako so se ljudje sporazumevali, ko še ni bilo telefonov, interneta, mesengerja … 
Vse to bomo spoznali pri urah zgodovine. 
Kajti zgodovina ni le naštevanje letnic, bitk in kraljev, ki zna biti včasih dolgočasno, temveč je živahna in pestra zgodba preteklih dni. 
Oseba, ki proučuje dogajanje v preteklosti, se imenuje zgodovinar. 
Lahko bi mu rekli tudi raziskovalec, ki proučuje in razlaga različne zgodovinske dogodke in pojave.

Opis gradnje prvih hiš na Bližnjem vzhodu.

Svetišča so gradili iz kamna in marmorja. 
Opremili so jih z dragimi kovinami (zlatom, bronom) in drugimi materiali (slonovino, dragim lesom). 
Pobarvali so jih v živopisnih barvah. 
V nasprotju z mogočnimi templji in veličastnimi gledališči so bile hiše Grkov skromne. 
V središču hiše je bilo odprto notranje dvorišče. 
V hišah je bilo več sob. 
Ženski prostori so bili ločeni od moških. 
Tekoče vode niso imeli, nekatere hiše so imele lasten vodnjak. 
Drugi so po vodo hodili k skupnim vodnjakom. 
Premožnejši so imeli manjše kopalnice. 
Pohištvo v povprečni grški hiši je bilo skromno. 
Stvari so hranili v košarah ali skrinjah. 
Grške hiše so bile grajene iz opeke. 
Ostrešje je bilo leseno in prekrito s strešniki. 
Tla so bila kamnita ali iz gline.

Zakaj so Rimljani gradili amfiteatre

MESTA: •Mesta so bila zavarovana z obzidjem, zaradi česar je primanjkovalo prostora, •hiše so bile zato ozke in visoke, •ker so bile večinoma lesene, so bili v mestih pogosti požari.

Gradbeništvo pred 200 leti ali kakšna so bila nova mesta

Obzidja so od nekdaj mesta varovala pred sovražniki. 
Z novim vekom se je v mesto selilo vedno več prebivalcev in število meščanov je hitro naraščalo. 
Ker so obzidja mestu preprečevala širjenje, so jih porušili. 
Mesta so se hitro širila, a brez načrtov, prometnih povezav in urejene kanalizacije. 
Postala so prenaseljena, hrupna in onesnažena. 
Priseljenci v mestih so iskali delo v tovarnah. 
Pred 200 leti so tovarne pomenile napredek, zato so se odpirale vedno nove in nove. 
V središčih mest so bile trgovine, banke, pisarne (sedeži podjetij). 
Tovarne so bile na obrobju mest. 
Nekoč srednjeveška mesta so se zelo spremenila in postala velemesta. 
Nekatera mesta so imela celo milijon in več prebivalcev. 
Še vedno pa je bilo opaziti veliko razliko med bivališči bogatih in revnih, med premožnimi lastniki tovarn in revnimi delavci.

NEMČIJA: PRVI AVTOMOBIL

Z razvojem človeštva je porasla menjava pridelkov in obrtnih izdelkov. 
Med bližnjimi in oddaljenimi vasmi, mesti ter državami je potekala živahna trgovina. 
Tovor so prenašali ljudje ali tovorne živali. 
Vse dokler niso v Mezopotamiji pred 7000 leti iznašli eno najpomembnejših iznajdb v zgodovini človeštva – kolo. 
Uporaba kolesa je pospešila hitrost potovanja in učinkovitost človekovega dela. 
Uporabljali so ga v gospodarstvu, prometu in kot del drugih pripomočkov.

V prvih skupnostih je bilo delo razdeljeno med moške in ženske.

Kmečka družina

Rokodelci delujejo tudi danes. 
Nekateri izdelujejo predmete in izdelke, ki so jih izdelovali že njihovi starši in stari starši. 
Mnogi rokodelci pa se ob uporabi novih gradiv preizkušajo v izdelovanju novih predmetov ali skušajo poiskati nove oblike znanih predmetov. 
Še vedno pa so rokodelski izdelki poznani po svoji lepoti in skrbnosti, s katero so izdelani. 
Večina ljudi v rokodelskih izdelkih ceni predvsem povezanost lepote in uporabnosti. 
Številni rokodelski izdelki so tudi primerna osebna, poslovna in protokolarna darila.
