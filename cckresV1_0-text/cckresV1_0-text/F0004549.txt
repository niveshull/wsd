Tlachtli (tlačtli) je sveta igra Aztekov. 
Azteki so verjeli, da tudi njihovi bogovi igrajo tlachtli. 
Igrišče bogov je nebo, sonce je njihova žoga. 
Azteki so uporabljali gumijasto žogo. 
Igrali so na dvorišču, obdanem z visokimi zidovi. 
Žoga ni smela pasti na tla. 
Udarjali so le z boki, koleni in komolci. 
Igro so jemali tako resno, da so se nekateri igralci poškodovali do smrti. 
Zmagovite voditelje moštev so zelo častili. 
Voditelje moštev, ki so igro izgubili, so žrtvovali bogu igre.
