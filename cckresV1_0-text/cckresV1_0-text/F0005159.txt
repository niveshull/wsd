Kako so nastale naše gore, jezera, reke? 
Kako je naš kraj dobil ime? 
Ljudje so od nekdaj iskali odgovore na takšna vprašanja, zato so se pripovedke o življenju nadnaravnih bitij ohranile do danes. 
Ljudje so pripovedovali, da so tako visoke gore lahko zgradili le velikani: »Velikani so bili torej staroselci v tej lepi deželi. 
Prišleki so jih imenovali ajdi. 
Predstavljali so si jih kot malce topoglave orjake, ki pa večinoma niso bili hudobni. 
Dali so ljudem celo ogenj in jih naučili uporabljati kladivo. 
Živeli so po gorah. 
Vsaka družina na svoji gori. 
Če so se hoteli med seboj kaj pogovoriti, so kričali z gore na goro. 
Če so bili žejni, so se nagnili z gore in zajemali vodo iz najbližje reke.« 
Poleg velikanov nastopajo v ljudskih bajkah in pripovedkah še zmaji, povodni možje, škrati, vile in žal žene. 
O njihovih nenavadnih dejanjih lahko beremo v knjigi Slovenske bajke in pripovedke. 
Zgodbe so že izhajale v reviji Otrok in družina, zbrala jih je Dušica Kunaver, ilustriral pa Matjaž Schmidt. 
Knjiga je izšla v samozaložbi. 
Tudi knjižna zbirka Zakladnica slovenskih pripovedi, ki jo je začela izdajati založba Didakta, želi otrokom približati bogastvo ljudske ustvarjalnosti. 
Delo Visoko v gorah, globoko v vodah je prva knjiga te zbirke. 
Zgodbe pripovedujejo o velikanih, vilah in povodnih možeh. 
Mag. Roberto Dapit in dr. Monika Kropej sta zbrala enaintrideset besedil iz različnih slovenskih pokrajin, tudi iz zamejstva. 
Zgodbe je z bogatimi ilustracijami dopolnil Veno Dolenc. 
Otroci se ob poslušanju vživijo v svet bajnih bitij in uživajo v bogati domišljiji zgodb. 
Ljudsko izročilo nam pove, kako so nekoč ljudje živeli in kako so si predstavljali svet. 
Včasih je bilo pripovedništvo namenjeno temu, da so se otroci naučili, kaj je dobro in prav. 
Tudi ob pravljicah najmlajši spoznavajo, kakšen je svet, in pomirja jih spoznanje, da dobro premaga zlo.
