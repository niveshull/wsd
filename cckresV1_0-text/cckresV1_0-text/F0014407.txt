LEPI KAMNI

Ljudje so slikali že v davnih časih. 
Papirja seveda takrat še niso poznali. 
Njihove risbe so nastajale na različnih naravnih podlagah. 
Na zgornji sliki so rečni kamni z zelo, zelo starimi risbami. 
Znanstveniki si še danes belijo glave, a ne morejo razvozlati, kaj pomenijo ti preprosti znaki. 

Kaj bi naslikali na kamne?

Poigrajmo se s črtami. 
Črte so lahko ravne ali krive. 
Ene so tanke, druge debele. 
Lahko so ostro zašiljene, lahko so nežno zaobljene.

MIŠKA ŠLA?

Kdo ve.

Dedek in babica sta odšla, račka pa se je spremenila v deklico. 
Perje je pustila v gnezdu, vstala je, se zasukala, izbo je pometla, prah je pobrisala, vode nanosila in kolačkov napekla.

»Pri vas je šepava deklica vode nanosila,« so jima rekli sosedje.

Kmalu je poletela nad hišo druga jata labodov.

S perjem se je deklica pokrila, postala je spet račka in je odletela.
