Kriki in renčanje so se spremenili v prve besede. 
Človek je lahko soljudem sporočal svoje misli. 
Tako se je začela pot kopičenja človekovega znanja. 
Svoje potomce je naučil vseh tistih spretnosti za življenje, ki so mu jih pokazali predniki, in dodal še svoje zamisli. 
Dokler je bilo znanja malo, je bilo povsem dovolj povedati in pokazati, kako se obdela kamen, kako zanetiš ogenj, izdelaš posodo ali ujameš medveda.

2. 
Mlini na veter so mleli žito ali črpali vodo. 
Najbolj znani so v zahodnem delu Evrope, kjer je dovolj stalnega vetra.

V starem Egiptu je bila medicina še posebno razvita. 
Zgradbo človeškega telesa so poznali zaradi balzamiranja, ki se je razvilo zaradi verovanja. 
Poznali so delovanje krvnega obtoka, pri operacijah so proti bolečinam uporabljali opij in opravljali kirurške ter zobozdravstvene posege. 
Obstajali so zdravniki specialisti za oči, glavo, zobe, trebuh, notranje bolezni … 
Pomagali so si s številnimi besedili, v katerih so bile popisane izkušnje prejšnjih generacij. 
Egipčanska medicina ni ubežala vplivu magije. 
Isti postopki zdravljenja so veljali za učinkovite ali neučinkovite, odvisno od dneva ali letnega časa uporabe postopka.

Otroštvo v antiki

V kmečkih družinah so otroci morali delati in pomagati preživeti svoji družini. 
Zelo težko so prišli do izobrazbe. 
V zelo redke samostanske šole so jih sprejeli kot pomoč pri delih ali pa so v njih videli bodočo podeželsko duhovščino. 
Če so imeli priložnost učiti se, se je njihov pouk razlikoval od pouka bogatejših učencev.

V: Albert G. Absenger/Walter Knarr/Herbert Pfeifer, Der Mensch im Wandel der Zeiten, Wien 1998, str. 190.

Ker je dejavnikov, ki ogrožajo kulturno dediščino, veliko, je za njeno ohranjanje pomembno sodelovanje različnih strokovnjakov: etnologov, zgodovinarjev, konservatorjev in restavratorjev ter vseh nas.

Do novega veka je bila najboljša zemlja namenjena pšenici, rži in ovsu. 
Od živinoreje je bila razvita predvsem ovčereja, ki je bila potrebna tudi za izdelavo oblačil. 
Manj donosne poljščine so nadomeščali z donosnejšimi, kot so krmne rastline, ajda, koruza, krompir, v Istri pa oljka, murva in vinska trta.
